# Convolution project using RUST language

# The theory of convolution
$X(\tau) = \int X(t) f(t) \dd{t}$


# Command to compile

### In debug mode
cargo build

cargo run

### Building for release version. recomended when deploying the app
cargo build --release

then you'll find an executable in the `target` folder.
